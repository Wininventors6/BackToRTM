# BackToRTM
BackToRTM is a Windows 10 mod to replicate the good ol' days of Windows 10 RTM! It is based off of Windows 10 LTSC 21H2, i specifically put LTSC because Windows 10 ended support
# Instructions
Download file from Releases.
Test in VM. (Don't use as ur main OS, buggy.)
# Why was this failure created?
Many people liked RTM better than all the other Windows 10 versions, saying it was the best.
# Features
NonXAML UAC
Old UWP apps (register using this command in powershell "add-appxpackage -register "C:\Program Files\WindowsApps\*\AppxManifest.xml" -disabledevelopmentmode")
Old start menu (can't remove hamburger menu)
Old Hero wallpaper
